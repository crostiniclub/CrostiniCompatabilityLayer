# Crostini Compatibility Layer Installer

Simple Bash installer for ChromeOS Crostini and Debian-based systems.

This script installs the compatibility layer file included with this repository.

---

# Installation

## Step 1: Download the repository

```bash
git clone https://github.com/yourusername/crostini-compat-layer.git
cd crostini-compat-layer
```

---

## Step 2: Make the installer executable

```bash
chmod +x installer.bash
```

---

## Step 3: Run the installer

```bash
./installer.bash
```

Or run with Bash:

```bash
bash installer.bash
```
