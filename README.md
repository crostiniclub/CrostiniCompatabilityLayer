# Crostini Compatibility Layer Installer

Simple Bash installer for ChromeOS Crostini and Debian-based systems.

This script installs the compatibility layer file included with this repository.

---

# Installation

## Step 1: Clone the layer 

```bash
git clone https://github.com/crostiniclub/CrostiniCompatabilityLayer.git
cd main.bash
```

---

## Step 2: Compile script

```bash
chmod +x main.bash
```

---

## Step 3: Run the script

```bash
./main.bash
```

Or run with Bash:

```bash
bash main.bash
```
